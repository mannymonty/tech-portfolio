# Mannymv — Portfolio

I build things that answer questions before anyone thinks to ask them — and ship them in a form that anyone on the team can actually use.

This repo covers seven projects across enterprise analytics, consumer fintech, process automation, pricing intelligence, and UX research tooling. Each one started with a real problem, a real user, and a gap nobody had filled. The through-line: I find where teams are losing time or missing signal, build the tool that fixes it, and make sure the output tells a clear story to whoever needs to act on it.
